chapterRowSelector

chapterRowSelector: string = "li.wp-manga-chapter"

Inherited from Madara.chapterRowSelector

  • Defined in src/Madara.ts:52

The standard HTML selector that denotes an individual chapter row is li.wp-manga-chapter. However, some sources have an alternate class or HTML element that denotes a row. The selector should return each row, meaning that one node returned by the selector corresponds to one chapter row, so this should return 0 rows if the chapter has no chapters and 1 or more rows if the chapter has 1 or more chapters.
